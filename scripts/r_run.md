\# How to Run Analysis.R



Requirements:

Install these R packages before running:

\- tidyverse

\- broom

\- ggpubr

\- readr



File structure (from repo root):

\- data/gpa.csv

\- data/results/   (outputs)

\- plots/          (outputs)



Run:

From project root in a shell:

Rscript scripts/Analysis.R



Or open scripts/Analysis.R in RStudio and click Source.



Outputs:

\- CSVs → data/results/

\- Plots → plots/
